# ThatMirror

## Run locally

```bash
npm start
```

Open:

`http://localhost:4200`

## VS Code folding

If a section does not show a folding arrow:

1. Select the section of code.
2. Press `Cmd + Shift + P`.
3. Search for `Create Manual Folding Range from Selection`.
4. Press Enter.

VS Code will add a folding arrow for the selected code.
